RocketBeans.TV Sendeplan App

Sprache: Swift
Plattform: Mac (OS X)

Mit dem Start des Twitch-Channels "RocketBeans.TV" habe ich eine unter MIT-Lizenz stehende App für Mac OS X entwickelt, die den aktuellen Sendeplan des Internet-Fernsehsenders RocketBeans.TV in der Menüleiste von OS X anzeigt. Bestandteil der App ist zudem ein Widget für die "Today"-Ansicht im OS X Notificationcenter sowie Push-Benachrichtigungen beim Start einer Sendung. Die App ist vollständig in der von Apple im Jahr 2014 vorgestellten Programmiersprache Swift entwickelt worden und ist mit 40.000 Backend-API-Calls pro Tag (Daten: Google Developer Console) eine meiner erfolgreichsten Apps.
